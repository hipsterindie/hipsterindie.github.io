Styles and Colors

We can style and color our HTML webpage. There is an entire other subject called CSS that can be used to do more with HTML. We will only work on the basics now. If we type "style" in our paragraph tag, we can color our text, as well as background color. By applying it to the body section of our code, we can change the entire color of the text, or in this case, the background. The other background color overrides the body's style.

Formatting

As we can see in the code, we have some new tags that involve organization. This helps allow code to be read easier or found quicker by yourself or some future developers, as well as being used in search engine optimization. Some more tags we've used are headers and footers, which usually contain nav tags, which allows for easy navigation throughout the website. there is also the main, which holds the majority of text. We use the article tag for the text of, let's say a blog post. This should(?) hold all our text of the blog or article. This can further be broken down with section tags. These section tags should have headers. We can use our header in a hierarchy fashion. This is only recommended, not required. We also have an aside tag that can be used for stuff not directly used on the page. Maybe an advertisement like Google AdSense. And again, this section is mainly for the layout of the HTML file, so these structure tags help make it easier to tell just what is going on. These aren't needed but it's a very handy set of tools. The basic idea with search engine optimization is that if our website has good search engine optimization, it makes it easier for search engines like Google to read through the site and figure out just what your website is about.
